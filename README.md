# TicTacToe

## Usage
`php app.php [-h | --help] [-g | --games=<gameCount>]`

## Output
`php app.php -g 2`

```
# Game 1, Draw

 O | X | O
-----------
 O | X | O
-----------
 X | O | X

X: 0 Y: 0 Draw: 1


# Game 2, O

   |   | X
-----------
 O | O | O
-----------
 O | X | O

X: 0 Y: 1 Draw: 1

Y Wins
```
